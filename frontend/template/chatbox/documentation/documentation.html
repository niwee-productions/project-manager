Introduction

InstaC, a modern and clean premium quality messaging template based on Bootstrap 4 using concept driven and minimal design. Its well optimized with mobile first responsive approach. InstaC built using SASS preprocessor, HTML5, CSS3 and help of usefull jQuery plugins.

Skin

You can use InstaC with dark or light mode. By default we use light mode.

To use InstaC in light mode, link dark-skin css file.

<link rel= "stylesheet" href= "./assets/skin/dark-skin.min.css" type= "text/css" >
